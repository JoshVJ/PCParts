PC Parts

Parts of a Computer
(In a Nutshell)

Quick Summary:

Computers all rely on these 6 basic parts:
The Motherboard
The Central Processing Unit (CPU)
The Random Access Memory (RAM)
The Graphics Processing Unit (GPU)
The Hard Disk Drive/Solid State Drive (HDD/SSD)
And the Power Supply Unit (PSU)

As you scroll down, there will be sections for each part to learn about.


The Motherboard:

The motherboard is a board that controls all of the major components of a computer, where you can find the CPU, GPU, RAM, and HDD. There are different motherboards, some specific for gaming, and some specific to business, yet all motherboards can hold components. These components, of course, are of different specs, as there are some components made for gaming and high-end computers, and some components made for office computers, or lower-end computers. Higher-end computers will need more expensive parts that can perform better and can handle more workload as opposed to lower-end computers.



The CPU

The CPU (also known as the Central Processing Unit) is commonly described as the "brains of the computer" as it controls everything that happens in the computer when run. It processes data in the computer's processes, and runs programs (which are instructions to the computer). Without a CPU, none of the other parts in a computer would work, as they all rely on the CPU to know what to do. CPUs work with the pins found at the bottom of the CPU, which interact with the motherboard to receive power and processes. The CPU goes in the socket in the motherboard in order to work with the rest of the components.



The RAM

The RAM (also known as Random Access Memory) stores short-term memory to be used by the CPU and the rest of the computer. The data stored is temporary, and resets when a computer reboots. RAM also can help out a CPU, as with faster RAM, the CPU can process faster. The difference between RAM, and HDDs/SSDs (which you will see soon) is that the storage is transferred instantly and is used directly by the CPU, it is not stored after a computer shuts down, as opposed to HDDs/SSDs which store the computer's storage permanently. RAM comes in the form of a stick which plugs into the motherboard, some RAM sticks are much flashier than other RAM (some having RGB lights), yet all RAM sticks have the same function, just with different power.



The GPU

The GPU (also known as the Graphics Processing Unit) processes graphics for the computer, and outputs them through HDMI, DisplayPort, or more rarely nowadays, VGA, allowing the user to use the computer and see what is happening. The GPU is really important when recording or in gaming, as it is the component which processes all of the graphics in the game, making the games run smoothly and look great. All computers need graphics cards, although some have integrated GPUs, which have the GPU integrated inside the CPU, good for those who don't need to record/game at high qualities. For more information on integrated GPUs (vs. Dedicated GPUs), check out this website.



The HDD/SSD

The HDD (also known as a Hard Disk Drive) stores permanent data onto a disk for the computer to use. The operating system (such as Windows or Linux) also boots with the help of a hard drive, or an SSD (Solid State Drive) which is much faster than a HDD, as it does not use a disk to store data and instead stores it with flash memory, similar to that of a USB stick. HDDs are cheaper and slower than SSDs, but typically store much more storage while SSDs are more expensive, but for fast memory. You can have multiple HDDs and SSDs in a PC, as long as you have space for it.



The PSU

The PSU (also called the Power Supply Unit) provides all power to the PC components, mainly the motherboard and hard drives, although in higher-end PCs with higher quality GPUs, some GPUs may take power directly from the power supply as well. There are pin connectors, which connect from the power supply to the component in order to provide power to the component, just like plugging in a charger to a wall, although with higher quality plugs. The PSU plugs into a regular outlet.
